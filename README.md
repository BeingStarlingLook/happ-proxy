**Happ** — универсальное приложение для работы с прокси-конфигурациями, использующее возможности ядра **Xray**. Оно объединяет поддержку современных сетевых технологий, гибкую систему настройки и удобный пользовательский интерфейс, позволяющий быстро создавать, импортировать и редактировать подключения независимо от уровня подготовки пользователя.

## Основные возможности

* Управление и редактирование прокси-конфигураций.
* Гибкая маршрутизация трафика с помощью настраиваемых правил.
* Совместимость с популярными протоколами:

  * **VLESS (Reality)**
  * **VMess**
  * **Trojan**
  * **Shadowsocks**
  * **SOCKS**
* Мгновенное переключение между несколькими профилями.
* Единый интерфейс для управления всеми подключениями.

## Конфиденциальность

Приватность пользователей является одним из основных принципов Happ. Приложение не собирает телеметрию, не передает персональные данные и не отправляет конфигурации на внешние серверы. Все настройки и служебная информация сохраняются исключительно на вашем устройстве.

## Важно знать

Happ предназначен исключительно для настройки и использования прокси-подключений. Приложение **не является VPN-сервисом**, не предоставляет собственных серверов и не продает доступ к ним. Для работы потребуется использовать собственную инфраструктуру либо сервер стороннего поставщика. Пользователь самостоятельно несет ответственность за соблюдение законодательства своей страны.
